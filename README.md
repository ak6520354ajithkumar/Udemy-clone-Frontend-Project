# Udemy Clone – HTML & CSS Project

This project is a basic **Udemy website clone** created using **HTML and CSS**. The goal of this project was to practice building responsive web pages and improve front-end development skills.

### Features

* Responsive navigation bar
* Search bar design
* Course categories section
* Recommended courses layout
* Clean and user-friendly interface
* Responsive design using Flexbox and CSS

### Technologies Used

* HTML5
* CSS3

### Purpose

This project helped me understand:

* Semantic HTML structure
* CSS styling and layout techniques
* Responsive web design principles
* Building real-world website layouts

**A simple Udemy clone built using HTML and CSS to practice responsive web design and front-end development skills.**
